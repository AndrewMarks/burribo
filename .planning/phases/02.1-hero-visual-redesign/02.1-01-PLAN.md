**Research-established patterns:**
- Pure CSS `transform: translate3d()` for GPU-accelerated animation
- `@keyframes` with `infinite linear` for seamless loop
- 3 stacked div layers with different animation speeds for parallax depth
- `overflow: hidden` container to mask tile repetition
- `will-change: transform` for performance optimization
- Absolutely-positioned clickable hotspots with pulse animation

**Asset sources from research:**
- itch.io: Free pixel art space backgrounds (CC0/Free)
- OpenGameArt: Parallax layer sets (CC0)
- Target: Minimal stars aesthetic, tileable, 3 depth layers

**Anti-patterns to avoid:**
- Do NOT animate `background-position` (causes repaints, use transform instead)
- Do NOT use JavaScript for continuous scroll animation (CSS is smoother)
- Do NOT exceed 3-4 parallax layers (performance)
- Do NOT use large unoptimized images (keep under 500KB total)
</context>
