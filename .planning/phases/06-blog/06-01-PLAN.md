<task type="auto">
  <name>Task 2: Add blog typography styles</name>
  <files>assets/css/main.scss</files>
  <action>
Add blog-specific styles to main.scss:

1. Post layout styles (.post):
   - Constrained content width (max-width: 65ch centered)
   - Generous padding for comfortable reading
   - Good line-height (1.7-1.8 for body text)

2. Post header (.post-header):
   - Title using --text-display but toned down (not uppercase)
   - Date/meta in muted color
   - Spacing to separate from content

3. Post content (.post-content):
   - Prose-optimized typography:
     - Comfortable font size (var(--text-base) or slightly larger)
     - Long-form line-height (1.7+)
     - Paragraph spacing (margin-bottom: 1.5em)
   - Heading styles (h2, h3) that work in prose:
     - NOT uppercase (override display font behavior)
     - Clear hierarchy with spacing
   - Link styles visible but not distracting
   - Code blocks if needed (use --color-bg-elevated)
   - Blockquote styling with left border accent

4. Blog index (.blog-index):
   - Post list with title, date, excerpt
   - Card-like styling but simpler than portfolio cards
   - Clear clickable areas

Use existing design tokens. Keep cohesive with site but prioritize readability over VALORANT boldness.
  </action>
  <verify>Read file and confirm blog styles added at end of main.scss</verify>
  <done>Blog typography styles complete with reading-optimized settings</done>
